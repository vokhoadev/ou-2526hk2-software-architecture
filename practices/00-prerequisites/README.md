# 00 - Prerequisites (Yêu cầu Chuẩn bị)

## Tổng quan

Thư mục này chứa hướng dẫn cài đặt môi trường cần thiết trước khi bắt đầu các labs thực hành.

## Thời lượng

**Tổng thời gian:** 5 giờ

| File | Nội dung | Thời gian |
|------|----------|-----------|
| environment-windows.md | Cài đặt trên Windows | 2h |
| environment-mac.md | Cài đặt trên macOS | 1.5h |
| cloud-accounts.md | Đăng ký tài khoản cloud | 1h |
| tools-setup.md | Cài đặt công cụ development | 1h |

## Checklist Hoàn thành

- [ ] Docker Desktop đã cài và chạy được
- [ ] VS Code với extensions cần thiết
- [ ] Git đã cấu hình
- [ ] Node.js 18+ đã cài
- [ ] Java 17+ đã cài
- [ ] AWS Free Tier account
- [ ] Azure Free account (optional)
- [ ] PlantUML extension hoạt động
- [ ] kubectl đã cài (cho Kubernetes labs)

## Yêu cầu Phần cứng

| Thông số | Minimum | Recommended |
|----------|---------|-------------|
| RAM | 8GB | 16GB |
| Disk | 30GB trống | 50GB trống |
| CPU | 2 cores | 4 cores |
| OS | Windows 10/11, macOS 11+, Ubuntu 20.04+ | - |
