# 01 - Foundations (Nền tảng Kiến trúc)

## Tổng quan

Level 1 cung cấp nền tảng về tư duy kiến trúc phần mềm, bao gồm phân tích quality attributes, trade-off analysis, và architecture views.

## Thông tin

| Thông tin | Giá trị |
|-----------|---------|
| **Level** | 1 - Beginner |
| **Số Labs** | 4 |
| **Thời lượng** | 12 giờ |
| **CLO đạt được** | CLO1, CLO2 |

## Danh sách Labs

| Lab | Tên | Thời lượng | Công cụ |
|-----|-----|-----------|---------|
| 1.1 | Quality Attributes Workshop | 3h | Miro, FigJam |
| 1.2 | Architecture Tradeoff Analysis | 3h | Decision Matrix |
| 1.3 | Architecture Principles (SOLID, DRY, KISS) | 3h | Code Review |
| 1.4 | Architecture Views (4+1 Model) | 3h | Draw.io, PlantUML |

## Mục tiêu Học tập

Sau khi hoàn thành Level 1, sinh viên có thể:

1. **Phân tích Quality Attributes** - Xác định và đánh giá các thuộc tính chất lượng của hệ thống
2. **Trade-off Analysis** - Phân tích đánh đổi giữa các quyết định kiến trúc
3. **Áp dụng Principles** - Hiểu và áp dụng SOLID, DRY, KISS, YAGNI
4. **Vẽ Architecture Views** - Sử dụng 4+1 View Model để mô tả kiến trúc
